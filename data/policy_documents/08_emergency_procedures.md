# Emergency Procedures Manual

## General Emergency Principles

### Priority Order
1. Aviate - Maintain aircraft control
2. Navigate - Know your position
3. Communicate - Inform ATC and cabin

### Crew Coordination
- Clear communication between cockpit and cabin
- Use standard phraseology
- Confirm all commands
- Brief when time permits

## Fire Emergencies

### Cockpit/Flight Deck Fire
1. Don oxygen masks
2. Identify source
3. Execute appropriate checklist
4. Consider emergency descent
5. Prepare for possible diversion

### Cabin Fire
1. Cabin crew fight fire with extinguishers
2. Cockpit notified immediately
3. Move passengers away from fire
4. Emergency descent if smoke in cabin
5. Land at nearest suitable airport

### Cargo Fire
1. Indicated by cargo fire warning
2. Execute cargo fire checklist
3. Land as soon as possible
4. Do not open cargo doors until fire services present

## Decompression Events

### Rapid Decompression
1. Don oxygen masks immediately
2. Emergency descent to 10,000 feet
3. Check passenger oxygen deployment
4. Assess structural damage
5. Divert to nearest suitable airport

### Slow Decompression
1. May be indicated by cabin altitude warning
2. Don oxygen masks
3. Controlled descent
4. Identify source if possible
5. Land at suitable airport

## Medical Emergencies

### Passenger Medical Emergency
1. Call for medical professionals on board
2. Use onboard medical kit
3. Contact MedLink ground support
4. Captain decides on diversion
5. Document all actions taken

### Crew Incapacitation
- Follow incapacitation checklist
- Secure incapacitated crew member
- Redistribute duties
- Consider diversion if critical position

### Medical Kit Contents
- Automated External Defibrillator (AED)
- Basic medications
- IV supplies (physician use only)
- Diagnostic equipment
- Emergency childbirth kit

## Security Threats

### Hijack/Unlawful Interference
1. Follow company security procedures
2. Use discrete notification codes
3. Comply while assessing options
4. Transponder code 7500
5. Follow ATC/security instructions

### Unruly Passenger
1. Attempt verbal de-escalation
2. Cabin crew restraint if necessary
3. Notify cockpit
4. Request law enforcement meet aircraft
5. Document incident thoroughly

### Bomb Threat
1. Treat all threats as credible
2. Notify captain immediately
3. Search procedures if time permits
4. Land as soon as practical
5. Follow ground security procedures

## Evacuation Procedures

### When to Evacuate
- Fire or smoke
- Structural damage
- Ditching
- Captain's command
- Unsafe to remain on board

### Evacuation Commands
- "EVACUATE, EVACUATE, EVACUATE"
- "UNFASTEN SEAT BELTS"
- "LEAVE EVERYTHING"
- "COME THIS WAY"

### Slide Deployment
1. Verify conditions outside
2. Arm doors as directed
3. Open door - slide auto-inflates
4. Assist passengers
5. Last crew member verifies cabin clear

### Ditching Procedures
1. Brief passengers on brace position
2. Distribute life vests
3. Position near overwing exits
4. Evacuate onto wings/rafts
5. Deploy life rafts

## Weather Emergencies

### Severe Turbulence
1. Reduce speed to turbulence penetration
2. Secure cabin
3. Passengers seated with belts fastened
4. Cabin service suspended
5. Deviate if possible

### Windshear
1. TOGA thrust
2. Follow flight director
3. Do not change configuration
4. Report to ATC
5. Consider go-around

### Volcanic Ash
1. Avoid known ash clouds
2. If encountered: reduce thrust, turn
3. Monitor engine parameters
4. Divert as necessary
5. Inspect aircraft post-flight

## System Failures

### Engine Failure
- Single engine: Follow checklist, continue or divert
- Dual engine: Highest priority emergency

### Electrical Failure
- Load shed non-essential
- Conserve battery
- Land as soon as practical

### Hydraulic Failure
- Identify failed system
- Use backup systems
- Prepare for abnormal landing

## Post-Emergency Actions

### Documentation
- Complete incident report within 24 hours
- Preserve CVR/FDR data
- Photograph damage if safe
- Collect witness statements

### Crew Care
- Critical Incident Stress debriefing
- Time off as needed
- EAP resources available
- No blame for good-faith actions
